## What is Linux

Linux is an operating system.
It helps computer to work properly.

Linux does these things:
- Manages files
- Manages users
- Manages memory
- Runs programs

Linux is used in many places:
- Servers
- Cloud systems
- Android mobiles
- ATM machines

## Terminal

Terminal is used to give commands to Linux.
It is like talking directly to the system.

GUI is helpful,
but RHCSA needs terminal knowledge.

---

## First Commands I Learned

pwd  
Shows where I am currently.

Alternative:  
echo $PWD

---

ls  
Shows files and folders in current location.

Alternative:  
ls -l  
ls -a

---

cd foldername  
Go inside a folder.

Alternative:  
pushd foldername

---

cd ..  
Go one step back.

Alternative:  
popd
